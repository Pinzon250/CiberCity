Frontend del Usuario Final
Responsabilidad: Todo lo que ve y usa el cliente en la tienda (comprar, navegar, registrar, valorar, etc.)
🧩 Módulos asignados:

Autenticación
    ✓ Registro / Login / Forgot password
    ✓ Sistema de JWT
    ✓ Protección de rutas

Catálogo público
    ✓ Listar productos con filtros (categoría, precio, marca, etc.)
    ✓ Búsqueda por nombre
    ✓ Ver detalles del producto

Carrito de compras
    ✓ Agregar/quitar productos
    ✓ Aplicar cupones
    ✓ Simulación de métodos de pago

Perfil del usuario
    ✓ Actualizar perfil
    ✓ Direcciones de envío
    ✓ Métodos de pago guardados
    ✓ Lista de deseos

Historial de compras y seguimiento de pedidos
    ✓ Ver órdenes anteriores
    ✓ Ver estado (procesando, enviado, entregado)
    ✓ Número de guía

Valoraciones y comentarios
    ✓ Crear valoración (estrellas + texto)
    ✓ Subida opcional de imagen

Soporte y contacto
    ✓ Formulario de contacto
    ✓ Mostrar preguntas frecuentes (FAQ)

Notificaciones y validaciones
    ✓ Confirmaciones por email

Facturación
    Visualizar detalles del pedido  
    Generar factura en PDF


Administrador / Gestión Interna
Responsabilidad: Control total de la tienda, gestión de productos, pedidos y monitoreo de ventas.

🧩 Módulos asignados:
Panel de administrador
    Login de administrador
    Dashboard con estadísticas
    Middleware para roles

Gestión de productos
    Crear, editar y eliminar productos
    Subida de imágenes
    Definir combos y categorías
    Marcar destacados, ofertas, nuevos
    Promociones especiales

Sistema de inventario
    Control de stock
    Mostrar “agotado”
    Controlar niveles bajos

Gestión de pedidos
    Ver pedidos por usuario
    Cambiar estados de pedido
    Asignar número de guía

Gestión de cupones
    Crear / editar / eliminar cupones
    Definir condiciones de uso

Gestión de preguntas frecuentes
    Crear y mantener las FAQs

Estadísticas y reportes
    Ventas por categoría
    Productos más vendidos
    Exportar reportes

Configuraciones generales
    Definir idiomas/monedas (si se habilita multilenguaje)
    Subir logo / branding desde backend
